### Syntax
**Program entry point**
```
fun main(){
	println("Hello world")
}
```

**Prints**
- Print without line break
```
print("Hello")
print("world")
________________________________
Hello world
```

- Print with line break
```
print("Hello")
print("world")
________________________________
Hello world
```

**Functions**
- Function with 2 Int type parameters and Int return type
```
// Addition function
fun sum(a: Int, b: Int): Int{
	return a + b
}
```
